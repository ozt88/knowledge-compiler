## unpatch_agent awk 동작 한계 (troubleshooting.md 참조)

install.sh의 `unpatch_agent` awk 로직:
```awk
/## Step 0: Knowledge Compile|## Step 10b: Knowledge Reconcile/ { skip=1 }
skip && /^## / && !/Knowledge Compile|Knowledge Reconcile/ { skip=0 }
!skip { print }
```

이 로직은 `## Step 0:` 또는 `## Step 10b:` 헤더로 시작하는 블록만 제거한다.
researcher와 planner 패치 블록은 `<!-- PATCH:knowledge-compiler -->` 주석으로 시작하며 `## Step N:` 헤더가 없다.
따라서 `install.sh --force`만으로는 researcher/planner 중복 블록이 제거되지 않는다.
verifier 패치는 `## Step 10b:` 헤더가 있으므로 `--force`로 처리 가능하다.
